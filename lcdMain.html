Useage

(1) Put this line in HTML. (ex. index.html) 
 <canvas id="canvasLCD"></canvas>

(2) Put this 2 lines in JavaScript (ex. main.js) 
 var lcd = new CanvasLCD('01');
 lcd.init('canvasLCD', 'initialword', true);
  • Line1 The augument is the color type. There are 6 types(01, 02, 03, 04, 05, 06). Those samples are in above.
  • Line2 The 1st augument is the ID of canvas in HTML, 2nd one is the first words after the LOGO disappears, 3rd one is the flag of displaying LOGO or not.
(3) Write some word in display. 
 lcd.write2Display('lettersRL01', 'ABCDEFG');
  • The augument is the display type([letters] display letters at once [lettersRL01] display each letter from left to right. [lettersRL02] display each letter from left to right. If the display has multi line, letters are displays one iteration; Please try demo in above.), 3rd one is the letters which displays in the LCD.
(*) The display digits also can be changed. 
var digits = { 'x': 18, 'y': 2 }; // Line55: canvasLCD.js

(*1) The display color also can be added. Add color set into colorSet object.
